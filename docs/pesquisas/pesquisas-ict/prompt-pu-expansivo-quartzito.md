# Prompt — PU Expansivo Imperveg (MAMONEX) e Quartzito como Agregado

## Objetivo

Pesquisar artigos científicos, patentes e relatórios técnicos sobre:

1. **PU Expansivo Imperveg (MAMONEX RD70)** — estudos de caracterização, aplicações estruturais, isolamento termoacústico e desempenho mecânico
2. **Quartzito como agregado em compósitos de PU vegetal** — uso de resíduo de quartzito como carga/agregado em matriz de poliuretano vegetal

## Dados já conhecidos

### MAMONEX RD70 (Imperveg)

| Parâmetro | Valor |
|-----------|-------|
| **Tipo** | Espuma rígida bicomponente de PU vegetal (óleo de mamona) |
| **Densidade** | 70 kg/m³ |
| **Resistência à compressão** | 0,25 MPa (10% deformação) — ficha técnica |
| **Potencial com agregados** | Fabio Takwara menciona até **200 MPa** em testes com agregados de quartzito |
| **Expansão** | ~10x o volume |
| **Gás de expansão** | CO₂ (sem CFC) |
| **Condutividade térmica** | 0,0452 W/mK |
| **Absorção de água** | 0,01% em 48h |
| **Estudo existente** | PIBIT Lucas Aciole + Ludmila Correia (UniCEUB, 2020) — painel acústico |

### Quartzito

- Mencionado por Fabio Takwara na reunião como agregado para PU vegetal, potencial de atingir **400 MPa**
- Resíduo de mineração — quartzito é uma rocha metamórfica composta essencialmente de quartzo
- Usado como carga mineral em compósitos poliméricos
