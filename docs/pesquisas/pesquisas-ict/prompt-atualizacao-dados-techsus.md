# Prompt — Atualização de Dados Técnicos do Sistema Faleiros-Techsus

## Objetivo

Pesquisar dados técnicos atualizados do Sistema Construtivo Industrializado Faleiros-Techsus (painéis nervurados pré-fabricados de concreto armado) e de seus concorrentes, para embasar a proposta FINEP Subvenção Econômica com referências pós-2020.

### Sistema
- **DATec Nº 024-B** — valido até out/2020
- Painéis de 140mm (42+56+42mm), concreto armado
- Patentes no Brasil, EUA e China

## O que buscar

### 1. Atualização dos ensaios IPT
- O IPT renovou o DATec 024-B? Existe um número novo (DATec Nº ...)?
- Há ensaios mais recentes do mesmo sistema construtivo (2008 vs. atual)?
- Ensaios de **inflamabilidade** para construção civil — quem faz no Brasil?

### 2. Normas técnicas vigentes
- NBR 15575 (Desempenho de edificações) — edição atual e requisitos
- NBR 6118 (Projeto de estruturas de concreto)
- NBR 9062 (Projeto e execução de estruturas de concreto pré-moldado)
- Atualizações pós-2020 que impactam o sistema

### 3. Concorrência — sistemas industrializados homologados CDHU
- Quais outros sistemas foram homologados no Chamamento CDHU Offsite 001/2024?
- Dados de desempenho comparativo (custo/m², tempo de obra, peso, isolamento)
- Empresas como: Tecverde, Systena, Woodframe, Steel frame, entre outros

### 4. Inovações em concreto com fibras vegetais
- Artigos com DOI sobre concreto com fibras vegetais (sisal, bambu, coco) pós-2020
- Estudos de ACV (Análise do Ciclo de Vida) de painéis de concreto industrializado
- Concreto com agregados reciclados (PET, borracha, resíduos de construção)

### 5. Patentes correlatas
- Patentes de sistemas de paredes duplas de concreto com núcleo oco (similares ao Techsus)
- Patentes de conectores para painéis pré-fabricados de concreto
### 6. PU vegetal em matriz cimentícia + PIBIT MAMONEX acústico

- Estudos com DOI sobre compatibilidade PU vegetal (mamona) + concreto/cimento
- Artigos do grupo do Prof. Khosrow Ghavami (PUC-Rio) e da USP São Carlos sobre impermeabilização de fibras vegetais para concreto
- **PIBIT Lucas Acioli + Ludmila Correia (CAU-DF/UNICEUB)** — ensaio de isolamento acústico da espuma expansiva MAMONEX RD70. Obter título, DOI, dados numéricos e PDF.

## Prioridade

**ALTA** — A base de dados de 2008 é insuficiente para a proposta FINEP. O edital exige demonstração de TRL e Incerteza Tecnológica baseada em dados atuais e literature review pós-2020.

## Formato de saída

Para cada item, retornar:
- Título completo, autores, ano, periódico/evento
- DOI (quando disponível)
- Dados numéricos relevantes (resistências, valores, normas)
- Link para o documento, quando público

Marcar como **NÃO ENCONTRADO** quando a busca não retornar resultado verificável.
